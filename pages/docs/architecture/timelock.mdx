# Timelocked Accounts

A timelock prevents an account from spending funds for specified period of time or block height.

To createa  timelock, you must provide the configuration when creating a new account. This can happen in the genesis ledger at the beginning of a network, or when creating a new account. In this section, we'll explore how to interact with a timelocked account and see how to create one ourselves.
