# SNARK и snarky

SNARK - это мощный криптографический инструмент, позволяющий использовать приложения.
которые дают пользователям беспрецедентный уровень контроля и конфиденциальности.
Они также являются ключевым элементом в создании масштабируемых децентрализованных систем, таких как Mina.

Для обзора на высоком уровне, ознакомьтесь с [этим блогом] (/blog/zero-knowledge-proofs-an-intuitive-explanation).

**snarky** - это собственный уникальный язык для использования SNARK.
Этот раздел предназначен в качестве руководства по началу использования SNARK в
ваших программах с использованием **snarky**.
